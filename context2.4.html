2.4.1 - Sequence, Selection, and Iteration

In programming, the flow of control is crucial in determining how the program executes. The three basic programming structures are:

Sequence

Sequence refers to executing statements one after another in a linear order. Each statement is executed in the order it appears in the code.

    # Example of Sequence
    x = 10
    y = 20
    z = x + y
    print(z)  # Output: 30

In the example above, the program executes each line of code sequentially: first, it assigns values to variables, then adds them, and finally prints the result.

Selection

Selection refers to choosing between different paths or actions based on conditions. This is typically done using if, else, and elif statements in many programming languages.

    # Example of Selection
    age = 18
    if age >= 18:
        print("You are an adult.")
    else:
        print("You are a minor.")

In this example, the program checks the condition if age >= 18. If the condition is true, the message "You are an adult" is printed; otherwise, the message "You are a minor" is printed.

Iteration

Iteration refers to repeating a block of code multiple times, which is commonly done with loops. There are two main types of loops in programming:

  • For loop: Repeats a block of code a specified number of times or over a sequence of values.
  • While loop: Repeats a block of code as long as a specified condition is true.
For Loop Example:
    # Example of For Loop
    for i in range(5):
        print(i)

The above code will print the numbers 0 to 4. The range(5) function generates numbers from 0 to 4, and the loop iterates over them, printing each number.

While Loop Example:
    # Example of While Loop
    i = 0
    while i < 5:
        print(i)
        i += 1  # Increment the value of i to avoid infinite loop

The while loop will continue as long as the condition i 5 is true. In this case, the loop will print the numbers 0 to 4.

In summary, sequence, selection, and iteration are fundamental building blocks of programming that determine how programs flow and execute. Sequence handles simple step-by-step execution, selection enables decision-making, and iteration provides the ability to repeat actions based on conditions or ranges.

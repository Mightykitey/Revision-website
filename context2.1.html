2.1.1 - Understanding Data Types

Data types define the kind of value a variable can hold. Understanding data types is fundamental for handling data in programming:

  • String: A sequence of characters, used to represent text. For example, "Hello, World!" is a string.
  • Character: A single letter, number, or symbol. For instance, 'a', '7', or '$' are all characters.
  • Integer: Whole numbers, positive or negative, without decimals. Examples: 42, -15, 0.
  • Real/Float: Numbers that contain a decimal point. For example, 3.14, -0.001, or 2.0.
  • Boolean: Represents one of two values: true or false. It’s commonly used for conditions in control structures.

2.1.2 - Declaring Constants and Variables

Variables and constants are used to store data. Variables can change, while constants retain their value throughout the program. Here’s how you declare them:

  • Variable declaration: int age = 30; (this declares a variable "age" of type integer with a value of 30)
  • Constant declaration: const double PI = 3.14159; (this declares a constant "PI" of type double with the value 3.14159)
  • String variable: string name = "John"; (this declares a variable "name" of type string with a value of "John")

2.1.3 - Data Structures

Data structures help us organize and store data efficiently. Different types of structures suit different tasks:

  • List: An ordered collection of items. Lists can store items of any data type, and they maintain the order of elements. Example: list numbers = {1, 2, 3};
  • Array: A collection of items stored at contiguous memory locations. Arrays store elements of the same data type and are fixed in size. Example: int[] scores = {90, 85, 88};
  • Dictionary: A collection of key-value pairs. Dictionaries allow for efficient lookups of values based on their corresponding keys. Example: Dictionary studentAges = {{"John", 22}, {"Alice", 20}};

2.1.4 - Managing Variables

Proper management of variables is essential for code readability, efficiency, and avoiding errors:

  • Local Variables: Variables declared inside a function or block, and only accessible within that scope. Example: int count = 10; // local to this function
  • Global Variables: Variables declared outside of all functions, accessible throughout the entire program. Example: int globalCounter = 100;
  • Variable Naming Conventions: Use meaningful names to describe the variable’s purpose, and follow naming conventions such as camelCase for variables (e.g., firstName) or snake_case (e.g., address_line_1) to improve readability.

2.2.1 - Mathematical Operators

Mathematical operators are used to perform basic arithmetic operations on numerical values. These are essential for performing calculations within programs:

  • Add: + - Adds two values. For example, 5 + 3 equals 8.
  • Subtract: - - Subtracts one value from another. For example, 5 - 3 equals 2.
  • Multiply: * - Multiplies two values. For example, 5 * 3 equals 15.
  • Divide: / - Divides one value by another. For example, 10 / 2 equals 5.0 (result is a float).
  • Integer Division: // - Divides and returns only the integer part of the result (ignoring the decimal). For example, 10 // 3 equals 3.
  • Modulus: % - Returns the remainder after division. For example, 10 % 3 equals 1.

2.2.2 - Relational Operators

Relational operators are used to compare two values and return a boolean value (true or false) based on the comparison:

  • Equal to: == - Checks if two values are equal. Example: 5 == 5 returns true.
  • Less than: < - Checks if the left value is less than the right value. Example: 3 < 5 returns true.
  • Greater than: > - Checks if the left value is greater than the right value. Example: 5 > 3 returns true.
  • Not equal to: <> - Checks if two values are not equal. Example: 5 <> 3 returns true.
  • Less than or equal to: <= - Checks if the left value is less than or equal to the right value. Example: 3 <= 5 returns true.
  • Greater than or equal to: >= - Checks if the left value is greater than or equal to the right value. Example: 5 >= 5 returns true.

2.2.3 - Boolean Operators

Boolean operators are used to perform logical operations on boolean values. These are frequently used in conditional statements to combine multiple conditions:

  • NOT: Negates a condition. If the condition is true, NOT makes it false, and vice versa. Example: NOT True returns false.
  • AND: Returns true if both conditions are true. Example: True AND True returns true, but True AND False returns false.
  • OR: Returns true if at least one condition is true. Example: True OR False returns true, and False OR False returns false.

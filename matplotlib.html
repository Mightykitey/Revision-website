matplotlib revision

Basic Plotting:

  • Line Plot: 
    import matplotlib.pyplot as plt
    x = [1, 2, 3, 4]
    y = [10, 20, 25, 30]
    plt.plot(x, y)
    plt.show()
    - Creates a simple line plot.
  • Scatter Plot: 
    plt.scatter(x, y)
    - Creates a scatter plot of x and y values.
  • Bar Plot: 
    plt.bar(x, y)
    - Creates a bar chart with x as categories and y as the height of the bars.

Customizing Plots:

  • Title: 
    plt.title("Plot Title")
    - Adds a title to the plot.
  • Labels: 
    plt.xlabel("X-axis label")
    plt.ylabel("Y-axis label")
    - Adds labels to the axes.
  • Grid: 
    plt.grid(True)
    - Adds a grid to the plot for better readability.

Saving Plots:

  • Save Plot to File: 
    plt.savefig("plot.png")
    - Saves the current plot to a file.

Subplots:

  • Create Subplots: 
    fig, axs = plt.subplots(2, 2)
    - Creates a 2x2 grid of subplots.
  • Access Individual Subplot: 
    axs[0, 0].plot(x, y)
    - Accesses a specific subplot in a grid.

Cross-Tabulation with Pandas:

  • Creating a Crosstab: 
    import pandas as pd
    data = pd.DataFrame({"Region": ["North", "South", "North", "East", "South"],
    "Salesperson": ["Alice", "Bob", "Alice", "Charlie", "Bob"],
    "Sales": [100, 200, 150, 120, 180]})
    crosstab_data = pd.crosstab(data["Region"], data["Salesperson"])
    - Creates a cross-tabulation of two categorical variables. In this case, `Region` and `Salesperson`.
  • Plotting Crosstab Data: 
    crosstab_data.plot(kind="bar", stacked=True)
    plt.title("Sales by Region and Salesperson")
    plt.xlabel("Region")
    plt.ylabel("Sales")
    plt.show()
    - Plots the cross-tabulated data as a stacked bar chart.

Example Code for Cross-Tabulation and Plotting:

# Example of Crosstab
    import pandas as pd
    data = pd.DataFrame({
        "Region": ["North", "South", "North", "East", "South"],
        "Salesperson": ["Alice", "Bob", "Alice", "Charlie", "Bob"],
        "Sales": [100, 200, 150, 120, 180]
    })
    
    # Creating a Crosstab
    crosstab_data = pd.crosstab(data["Region"], data["Salesperson"])
    print(crosstab_data)

# Example of Plotting Crosstab Data
    crosstab_data.plot(kind="bar", stacked=True)
    plt.title("Sales by Region and Salesperson")
    plt.xlabel("Region")
    plt.ylabel("Sales")
    plt.show()

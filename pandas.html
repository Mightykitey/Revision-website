pandas revision

Reading Data:

  • Read CSV: 
    import pandas as pd
    data = pd.read_csv("file.csv")
    - Reads a CSV file into a DataFrame. If the file exists and the format is correct, it loads the data into a DataFrame object.
  • Read Excel: 
    data = pd.read_excel("file.xlsx")
    - Reads an Excel file into a DataFrame. You can specify the sheet name with the sheet_name parameter.
  • Read SQL: 
    import sqlite3
    conn = sqlite3.connect("database.db")
    data = pd.read_sql("SELECT * FROM table", conn)
    - Reads data from a SQL database into a DataFrame using a SQL query.

DataFrame Operations:

  • Display First Rows: 
    data.head()
    - Returns the first 5 rows of the DataFrame. Useful for quickly inspecting the data.
  • Display Last Rows: 
    data.tail()
    - Returns the last 5 rows of the DataFrame.
  • Columns: 
    data.columns
    - Lists all the column names of the DataFrame.
  • Filtering Data: 
    filtered_data = data[data['column_name'] > 10]
    - Filters rows where values in a column meet a certain condition.

Data Manipulation:

  • Adding a New Column: 
    data['new_column'] = data['existing_column'] * 2
    - Creates a new column by performing operations on existing columns.
  • Dropping a Column: 
    data = data.drop('column_name', axis=1)
    - Removes a column from the DataFrame.
  • Renaming Columns: 
    data = data.rename(columns={'old_name': 'new_name'})
    - Renames columns in the DataFrame.

Handling Missing Data:

  • Check for Missing Values: 
    data.isnull().sum()
    - Returns a count of missing (NaN) values in each column.
  • Drop Missing Values: 
    data = data.dropna()
    - Removes rows with missing values.
  • Fill Missing Values: 
    data = data.fillna(0)
    - Fills missing values with a specified value (e.g., 0).

Grouping Data:

  • Group by Column: 
    grouped_data = data.groupby('column_name').sum()
    - Groups the DataFrame by a specific column and applies a sum operation to each group.
  • Multiple Aggregations: 
    grouped_data = data.groupby('column_name').agg({'column_to_agg': ['sum', 'mean']})
    - Applies multiple aggregation functions to a specific column after grouping.

Data Export:

  • Export to CSV: 
    data.to_csv("output.csv", index=False)
    - Saves the DataFrame to a CSV file. The index=False parameter avoids saving the index as a column.
  • Export to Excel: 
    data.to_excel("output.xlsx", index=False)
    - Saves the DataFrame to an Excel file.
  • Export to SQL: 
    data.to_sql('table_name', conn, if_exists='replace')
    - Saves the DataFrame to a SQL database, replacing the existing table if it exists.

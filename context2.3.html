2.3.1 - Using Text Files for Input and Output

In programming, text files are often used to store and retrieve data. This allows programs to handle data that doesn't need to be hardcoded in the source code. There are basic operations you can perform on text files: opening, reading, writing, and closing files.

Opening Files

To work with a file, it must first be opened. The open() function is used to open a file in different modes:

  • Read mode: open("file.txt", "r") - Opens the file for reading. If the file does not exist, it raises an error.
  • Write mode: open("file.txt", "w") - Opens the file for writing. If the file doesn't exist, it creates a new file. If it does exist, it overwrites the content.
  • Append mode: open("file.txt", "a") - Opens the file for appending. If the file doesn't exist, it creates a new file. This mode does not overwrite existing content.

Reading from Files

After opening the file in read mode, you can read the contents of the file using functions like:

  • read(): Reads the entire file as a single string. Example: content = file.read().
  • readline(): Reads one line at a time. Example: line = file.readline().
  • readlines(): Reads all lines into a list. Example: lines = file.readlines().

Writing to Files

To write to a file, you need to open the file in write or append mode. You can use methods like:

  • write(): Writes a string to the file. Example: file.write("Hello, World!").
  • writelines(): Writes a list of strings to the file. Example: file.writelines(["line 1", "line 2"]).

After performing the necessary operations on a file, it is important to close the file using the close() method. This ensures that all changes are saved and resources are released:

  • file.close() - Closes the file after operations are completed.

Example Code for File Handling

    # Example of reading from a file
    with open("file.txt", "r") as file:
        content = file.read()
        print(content)

    # Example of writing to a file
    with open("file.txt", "w") as file:
        file.write("Hello, this is a text file.")

In the example above, the with statement is used for file handling, which automatically takes care of closing the file after the operations are done, even if errors occur during the process.
